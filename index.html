Open Toonz Portable Edition

OpenToonz Portable Edition is just what it sounds like. You can use the powerful OpenToonz 2D animation software without installing it. Getting up and running is simple:

  • Download the zip file
  • Unzip the zip file to a destination of your choice
  • Run the OpenToonz program
  • Make awesome things.

Releases

The main release tracks with the current master branch of OpenToonz. New builds will be posted every couple of days based on additions to the master branch.

If there is interest in a bleeding edge version, with features that haven't yet been pulled into the master branch, please let me know and I will try and post a version.

Problems and Bugs

This software has not been thoroughly test yet. There are bound to be bugs. Please submit issues as you find them and I will try and get to them
